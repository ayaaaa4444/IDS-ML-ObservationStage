# IDS-ML-ObservationStage

## Description
Stage d'observation de première année – Découverte des systèmes de détection
d'intrusion (IDS) et initiation au Machine Learning appliqué à la cybersécurité.

**Entreprise d'accueil :** TERABYTE SOFTWARE  
**Période :** 20 juin – 20 juillet 2026  
**Réalisé par :** Aya MOHHI  

---

## Objectif
Comprendre le fonctionnement des IDS, observer les pratiques de l'entreprise
et réaliser une initiation pratique sur le jeu de données NSL-KDD.
